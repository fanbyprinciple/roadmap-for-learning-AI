## Weekly Study Structure

Daily commitment: 2–3 hours

* 45 minutes: Concepts
* 60 minutes: Hands-on coding
* 30 minutes: Debugging and review
* 15 minutes: Documentation

---

## Week 1: Agentic AI Foundations

* Monday: LLM basics, prompts, function calling
* Tuesday: LangChain fundamentals
* Wednesday: Introduction to multi-agent systems
* Thursday: LangGraph architecture
* Friday: Implement a basic multi-agent workflow
* Saturday: Mini-project – two-agent research assistant
* Sunday: Review and documentation

## Week 2: Agent Orchestration

* Monday: CrewAI fundamentals
* Tuesday: CrewAI advanced workflows
* Wednesday: AutoGen and Swarm architecture
* Thursday: Tool calling and memory
* Friday: Build a three-agent pipeline
* Saturday: Mini-project – task execution agent
* Sunday: Review and documentation

* Monday: Query rewriting and transformation
