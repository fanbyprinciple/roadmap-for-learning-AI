# Overview

This challenge is structured into eight weeks, each with clear deliverables, hands-on projects, and evaluation checkpoints. By the end, you will have portfolio-ready, enterprise-grade projects demonstrating your capability as an Agentic AI Engineer.

---

# Week 1: Python, Machine Learning, and LLM Fundamentals

## Goals

* Refresh advanced Python and machine learning foundations.
* Build a strong understanding of transformer models and tokenization.

## Key Topics

* Advanced Python
* Machine learning and deep learning fundamentals
* Attention and transformer architecture
* Environment setup

## Deliverables

* Notes summary
* Two small coding exercises (parsers and embeddings)
* Transformer-from-scratch mini implementation (optional)

---

# Week 2: LLM Frameworks and Vector Databases

## Goals

* Learn to use LangChain, LlamaIndex, and Haystack.
* Work with vector databases.

## Key Topics

* Chains, tools, and agents
* Document loaders and retrievers
* Embeddings and indexing

## Deliverables

* Mini project: simple query engine
* Vector database benchmark comparison
* Repository with notebooks and scripts

---

# Week 3: RAG Fundamentals

## Goals

* Build a basic retrieval-augmented generation (RAG) pipeline from scratch.

## Key Topics

* Chunking and metadata
* Retrieval strategies
* Reranking fundamentals

## Deliverables

* Full RAG pipeline
* Retrieval evaluation report
* Small REST API for the RAG system

---

# Week 4: Advanced RAG and Reranking

## Goals

* Build a hybrid search and reranker RAG pipeline.

## Key Topics

* BM25 and vector retrieval
* Cross-encoder reranking
* Query rewriting

## Deliverables

* Hybrid RAG system
* Performance benchmark notebook
* API endpoints for search and reranking

---

# Week 5: Multi-Agent Systems

## Goals

* Build systems using CrewAI, LangGraph, or Autogen.

## Key Topics

* Agent creation
* Task orchestration
* Tool integration
* Routing and evaluation

## Deliverables

* Multi-agent workflow (research, coding, and evaluator agents)
* Architecture diagram
* Execution logs

---

# Week 6: Fine-Tuning LLMs

## Goals

* Fine-tune a small model in the 3B to 7B range.
* Evaluate and benchmark the model.

## Key Topics

* LoRA, QLoRA, and PEFT
* Dataset preparation
* Evaluation metrics (ROUGE, BLEU, Ragas)
* Inference optimization

## Deliverables

* Fine-tuned model
* Evaluation report
* Model card

---

# Week 7: Deployment and MLOps

## Goals

* Deploy your systems and configure monitoring.

## Key Topics

* FastAPI deployment
* Docker and containers
* VLLM and TGI model serving
* Monitoring with OpenTelemetry and Prometheus

## Deliverables

* Deployed RAG API
* Dockerfile and container build
* Monitoring dashboard

---

# Week 8: Enterprise-Grade Capstone

## Goals

Combine all your work into a production-ready portfolio project.

## Key Topics

* Multi-agent orchestration
* Advanced RAG
* Fine-tuned model
* CI/CD
* Logging and monitoring

## Deliverables

* End-to-end enterprise AI platform
* Documentation and architecture diagrams
* Recorded walkthrough or README
* GitHub repository

---

# Final Output

By day 60, you will have:

* A production-grade multi-agent system
* A fully optimized RAG pipeline
* A fine-tuned LLM for a specific domain
* A deployable API with observability
* A polished capstone project demonstrating GenAI engineering capability
